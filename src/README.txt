Functional programming
------------------------
Створіть проект Functional-task, де реалізуйте відповідний
функціонал.
+
На платформі зареєстровано 5-ть користувачів.
Користувач має властивості: ім'я, телефон.
+
Сформуйте вихідні дані та виведіть їх в консоль.
+
Використайте в проекті можливості java.util.function.Consumer.
+
Забезпечте розподілення функціоналу проекту по відповідним класам.
+
